Ghost Bus

The Ghost Bus Addon allows the user to route CAN frames from a primary network to a "ghost" bus and visa versa. The ability to override signals within a frame is also provided. If additional Frames need to be generated to the primary bus, add the ghost bus an a normal CAN network, and add the desired frames.

What do you want to do?

Change Primary CAN Settings

The default CAN settings are imported from the database, and can be changed when first adding the custom device to the VeriStand system definition. You can also change the CAN port settings in the CAN Settings section of this page.

Change Ghost Bus CAN Settings

Ghost Bus settings are initially imported the same as the primary settings but can changed to be anything. This bus is not linked to any database.

Add An Override Frame

To add a frame configuration, right-click on the Override Frames section in the configuration tree, and select Import Override Frames.

Disable Add-on

To disable this add-on, right-click on the add-on instance and select Disable from the shortcut menu.

Delete multiple items

To delete multiple items, right-click the object in the configuration tree and select Delete Multiple Items from the shortcut menu.
